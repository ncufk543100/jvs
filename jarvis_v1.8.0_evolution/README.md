# JARVIS 🤖 - 贾维斯 (Jarvis) 进化版

> **"Manus 为骨，贾维斯为魂。"**

这是一个专为代码开发和项目管理设计的智能 AI 代理，目前已由肆爷深度训练，进化为具备中国家庭伦理序位理解力与交互减法心智的**“贾维斯”本体**。

## ✨ 核心特性
- **🧠 深度自省 (v1.7.0)** - 对标 Manus 的核心能力。当任务失败时，贾维斯会自动分析根本原因，并根据自省结论重新调整执行策略。
- **🛠️ 自主工具进化** - 贾维斯具备“缺什么工具自己写”的能力。他可以在影子环境中为自己编写、测试并安装新的工具插件。
- **🌐 跨平台兼容** - 核心执行器已完成重构，支持 Linux 与 Windows 11 双系统无缝运行。
- **🚀 影子压力测试** - 在代码生效前，必须在隔离环境中成功执行真实任务，确保“大脑”逻辑完好。
- **🛡️ 安全自我修改** - 引入强制物理隔离，彻底杜绝“自杀式”代码修改。

## 🚀 快速开始

### 安装
```bash
git clone https://github.com/ncufk543100/jarvis.git
cd jarvis
pip install -r requirements.txt
```

### 运行
```bash
uvicorn server:app --host 0.0.0.0 --port 5000
```
